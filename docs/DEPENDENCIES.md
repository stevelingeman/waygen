# Waygen Dependencies

This document lists the key dependencies used in the Waygen project.

## Production Dependencies

### Mapping & Geospatial
- **mapbox-gl** (^2.15.0) - Core mapping library
- **@mapbox/mapbox-gl-draw** (^1.4.3) - Drawing tools for Mapbox
- **@mapbox/mapbox-gl-geocoder** (^5.0.1) - Location search functionality
- **mapbox-gl-draw-circle** (^1.1.2) - Custom draw mode for circles
- **@turf/turf** (^6.5.0) - Geospatial analysis library

### File Processing
- **@tmcw/togeojson** (^5.8.1) - KML/KMZ to GeoJSON conversion
- **jszip** (^3.10.1) - ZIP file creation for KMZ export
- **file-saver** (^2.0.5) - Client-side file downloads

### UI & React
- **react** (^18.2.0) - UI framework
- **react-dom** (^18.2.0) - DOM rendering for React
- **lucide-react** (^0.292.0) - Icon library
- **zustand** (^4.4.6) - State management

## Development Dependencies

### Build Tools
- **vite** (^5.0.0) - Build tool and dev server
- **@vitejs/plugin-react** (^4.2.0) - React support for Vite
- **vite-plugin-commonjs** (^0.10.1) - CommonJS compatibility

### Styling
- **tailwindcss** (^3.3.5) - Utility-first CSS framework
- **autoprefixer** (^10.4.16) - CSS vendor prefixing
- **postcss** (^8.4.31) - CSS processing
